## Projects

### EarSense (Final Year Project)
- Designed an innovative generic dry electrode earpiece for collecting ear-EEG data
- Ag/AgCl palette electrodes within the ear canal with active shielding
- Fabricated using Luxaprint (hearing aid material)
- Validated using auditory mismatch negativity (MMN) paradigm
- Achieved correlation coefficients of 0.89, 0.85, and 0.71 with scalp EEG
- Reduced impedance to ~250 mΩ (from 3 kΩ in previous attempts)
- Implemented Brain-Computer Interface using P300 signals and Donchin speller matrix
- Used Grad-CAM for interpretability
- Achieved 0.8 accuracy with AlexNet and 0.83 with ResNet50
- Class II Medical Device (IEC 60601-1), Type BF Applied Part
- WiFi-enabled, battery powered (3.7V Li-Ion)
- **Award:** Best Undergraduate Thesis Project in Signal Processing Domain (IEEE Signal Processing Society, Nov 2024)
- **Team:** Lahiru Shyamal, Yohan Abeysinghe, Lakshan Rathnayake
- **Supervisors:** Dr. Anjula De Silva, Dr. Chamira Edussooriya
